# GitHub Pages Setup Guide

## 🚀 How to Enable GitHub Pages

### Step 1: Go to Your Repository Settings
1. Visit: https://github.com/xrstage-hyl/xrs-homepage-2025
2. Click on the **"Settings"** tab (near the top of the page)

### Step 2: Navigate to Pages Section
1. In the left sidebar, click on **"Pages"**
2. You should see a section called "Build and deployment"

### Step 3: Configure Source
1. Under **"Source"**, select **"Deploy from a branch"**
2. Under **"Branch"**, select **"main"** from the dropdown
3. Under **"Folder"**, select **"/ (root)"**
4. Click **"Save"**

### Step 4: Wait for Deployment
- GitHub will automatically build and deploy your site
- This usually takes 1-2 minutes
- You'll see a green checkmark when it's ready

### Step 5: Access Your Live Website
- Your website will be available at: `https://xrstage-hyl.github.io/xrs-homepage-2025`
- You can also find this URL in the Pages settings

## 🎯 What This Does
- Makes your XR Stage website live on the internet
- Provides a public URL anyone can visit
- Automatically updates when you push changes to GitHub
- Free hosting provided by GitHub

## 📝 Next Steps After Setup
1. Test your live website
2. Share the URL with others
3. Make updates by pushing new commits to GitHub
4. Consider adding a custom domain (optional)

## 🔧 Troubleshooting
- If you see a 404 error, wait a few minutes for deployment
- Make sure all files are in the root directory
- Check that `index.html` is in the main branch
